# Projet Data Analyst – Analyse de données E-commerce

Projet d’analyse de données e-commerce réalisé dans le cadre de mon apprentissage en data analytics, en utilisant Excel, SQL et Python.

## Description du projet

Ce projet consiste à analyser un jeu de données e-commerce afin de mieux comprendre les ventes, les performances des produits et la répartition des clients. L’objectif est de transformer des données brutes en informations exploitables.

## Objectifs

* Nettoyer et préparer les données
* Analyser les ventes et les comportements des clients
* Identifier des tendances dans les données
* Structurer les résultats pour faciliter leur interprétation

## Outils utilisés

* Excel pour l'exploration et prétraitement des données
* SQL Server pour extraction et manipulation des données
* Python - Pandas pour l'analyse et traitement des données

## Structure du projet

* data/ : données brutes
* excel/ : préparation et premières analyses
* sql/ : requêtes SQL pour l’extraction et l’analyse
* notebooks/ : analyses réalisées avec Python

## Démarche méthodologique

### Collecte des données

Les données ont été importées et organisées afin de pouvoir être exploitées dans différents outils.

### Nettoyage des données

Une étape importante du projet a été le nettoyage des données, notamment :

* suppression des valeurs manquantes
* correction des types de données
* gestion des erreurs de format

Cette étape m’a permis de mieux comprendre l’importance de la qualité des données avant toute analyse.

### Analyse des données

L’analyse a porté sur plusieurs aspects :

* les ventes globales
* les performances des produits
* la répartition des ventes par pays et dans le temps

### Visualisation

Les résultats ont été structurés sous forme de tableaux et d’analyses permettant de mieux comprendre les tendances observées.

## Résultats

L’analyse a permis de mettre en évidence plusieurs éléments intéressants :

* certains produits génèrent une part importante du chiffre d’affaires
* les ventes ne sont pas réparties de manière homogène selon les pays
* certaines périodes présentent une activité plus élevée

## Compétences mobilisées

* Manipulation et interrogation de bases de données (SQL)
* Nettoyage et préparation de données
* Analyse exploratoire
* Utilisation de Python pour le traitement de données
* Interprétation des résultats

## Perspectives d’amélioration

* Approfondir l’analyse du comportement client
* Ajouter des analyses plus avancées
* Optimiser les traitements de données

## Conclusion analytique

Ce projet m’a permis de mettre en pratique plusieurs compétences en analyse de données, notamment le nettoyage, l’exploration et l’interprétation des données.
Il m’a également montré que la qualité des données est essentielle pour obtenir des résultats fiables. Enfin, cette expérience m’a aidée à mieux comprendre comment structurer une analyse pour répondre à des questions concrètes.
