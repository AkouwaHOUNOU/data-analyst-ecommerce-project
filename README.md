End-to-end data analysis project using Excel, SQL, Python and Power BI

# E-commerce Data Analysis Project
## Objective
The objective of this project is to analyze an e-commerce dataset in order to identify sales trends, understand customer behavior, and evaluate performance across different countries.
## Tools Used
- Microsoft Excel (data cleaning, transformation, and exploratory analysis)
- SQL (data querying and aggregation)
- Python - Pandas (data processing and analysis)
- Power BI (data visualization and dashboard creation)
## Project Structure
- data/ : raw dataset used for analysis
- notebooks/ : Python notebooks for data processing and analysis
- sql/ : SQL queries for data extraction and aggregation
- powerbi/ : Power BI dashboard files
- excel/ : Excel files used for data cleaning and analysis
## Project Status
This project is currently in progress as part of my learning and practical application in data analytics.
